Driver photos folder

Place one image per driver using this naming pattern:
- lowercase
- spaces and punctuation replaced with '-'

Examples:
- max-verstappen.png
- lewis-hamilton.jpg
- charles-leclerc.webp
- michael-schumacher.png

Supported formats: .png, .jpg, .jpeg, .webp

If no file is found, the app shows an automatic initials avatar fallback.
